Golab’s theorem


Theorem. Let D be the unit disc of a Minkowski plane and let ℓ⁢(∂⁡D) denote the Minkowski length (http://planetmath.org/LengthOfCurveInAMetricSpace) of the boundary of D. Then 6≤ℓ⁢(∂⁡D)≤8. The lower bound is attained if and only if D is linearly equivalent to a regular hexagon. The upper bound is attained if and only if D is a parallelogram.

Note that 1/2 the perimeter of the unit disc is a constant between 3 and 4. The special case of the 2-norm yields a constant, which is known as π. So Golab’s theorem is that ”pi” for a Minkowski plane is always between 3 and 4.
