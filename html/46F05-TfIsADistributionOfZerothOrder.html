Tf is a distribution of zeroth order


To check that Tf is a distribution of zeroth order (http://planetmath.org/Distribution4), we shall use condition (3) on this page (http://planetmath.org/Distribution4). First, it is clear that Tf is a linear mapping. To see that Tf is continuous, suppose K is a compact set in U and u∈𝒟K, i.e., u is a smooth function with support in K. We then have

|Tf⁢(u)| = |∫Kf⁢(x)⁢u⁢(x)⁢𝑑x|
≤ ∫K|f⁢(x)|⁢|u⁢(x)|⁢𝑑x
≤ ∫K|f⁢(x)|⁢𝑑x⁢||u||∞.

Since f is locally integrable, it follows that C=∫K|f⁢(x)|⁢𝑑x is finite, so

|Tf⁢(u)|≤C⁢||u||∞.

Thus f is a distribution of zeroth order ([1], pp. 381). □
