special elements in a C*-algebra and their spectral properties


Definition - Suppose 𝒜 is a C*-algebra (http://planetmath.org/CAlgebra). An element x∈𝒜 is said to be:

  • •

    normal if x*⁢x=x⁢x*

  • •

    self-adjoint if x*=x

  • •

    unitary if 𝒜 has an identity element e and x*⁢x=x⁢x*=e

  • •

    positive if x=y*⁢y for some element y∈𝒜

  • •

    projection if x*=x and x2=x

  • •

    partial isometry if x*⁢x and x⁢x* are both projections

0.0.1 Properties of the special elements in terms of their spectrum

In the following σ⁢(x) denotes the spectrum of an element x and Rσ⁢(x) its spectral radius.

Theorem 1 - Suppose 𝒜 is a C*-algebra and x∈𝒜. If x is normal then ∥x∥=Rσ⁢(x)

Theorem 2 - Suppose 𝒜 is a C*-algebra and x∈𝒜.

  • •

    If x is self-adjoint, then σ⁢(x)⊂ℝ.

  • •

    If x is unitary, then σ⁢(x)⊂∂⁡D, where D⊂ℂ is the unit disk.

  • •

    If x is positive, then σ⁢(x)⊂ℝ+

  • •

    If x is a projection, then σ⁢(x)⊂{0,1}

Theorem 3 - Suppose 𝒜 is a commutative C*-algebra and x∈𝒜. Then

  • •

    x is self-adjoint if and only if σ⁢(x)⊂ℝ.

  • •

    x is unitary if and only if σ⁢(x)⊂∂⁡D, where D⊂ℂ is the unit disk.

  • •

    x is positive if and only if σ⁢(x)⊂ℝ+

  • •

    x is a projection if and only if σ⁢(x)⊂{0,1}

Theorem 4 - Suppose 𝒜 is a C*-algebra and x is normal in 𝒜. Then
