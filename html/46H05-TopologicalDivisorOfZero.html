topological divisor of zero


Let A be a normed ring.  An element a∈A is said to be a left topological divisor of zero if there is a sequence an with  ∥an∥=1  for all n such that

limn→∞⁡∥a⁢an∥=0.

Analogously, a is a if

limn→∞⁡∥bn⁢a∥=0,

for some sequence bn with  ∥bn∥=1.  The element a is a topological divisor of zero if it is both a left and a topological divisor of zero.

Remarks.

  • •

    Any zero divisor is a topological divisor of zero.

  • •

    If a is a (left) topological divisor of zero, then b⁢a is a (left) topological divisor of zero. As a result, a is never a unit, for if b is its inverse, then 1=b⁢a would be a topological divisor of zero, which is impossible.

  • •

    In a commutative Banach algebra A, an element is a topological divisor of zero if it lies on the boundary of U⁢(A), the group of units of A.
