commutant is a weak operator closed subalgebra


Let H be a Hilbert space and B⁢(H) the algebra of bounded operators in H. Recall that the commutant of a subset ℱ⊂B⁢(H) is the set of all bounded operators that commute with those of ℱ, i.e.

ℱ′:={T∈B⁢(H):T⁢S=S⁢T,∀S∈ℱ}.

- If ℱ⊂B⁢(H), then ℱ′ is a subalgebra of B⁢(H) that contains the identity operator and is closed in the weak operator topology.

: It is clear that ℱ′ contains the identity operator, since it commutes with all operators in B⁢(H) and in particular with those of ℱ.

Let us now see that ℱ′ is a subalgebra of B⁢(H). Let T1,T2∈ℱ′ and λ∈ℂ. We have that, for all S∈ℱ,

S⁢(T1+T2)=S⁢T1+S⁢T2=T1⁢S+T2⁢S=(T1+T2)⁢S
S⁢(λ⁢T1)=λ⁢S⁢T1=λ⁢T1⁢S
S⁢(T1⁢T2)=T1⁢S⁢T2=T1⁢T2⁢S

thus, T1+T2, λ⁢T1 and T1⁢T2 all belong to ℱ′, and therefore ℱ′ is a subalgebra of B⁢(H).

It remains to see that ℱ′ is weak operator closed. Suppose (Ti) is a net in ℱ′ that converges to T in the weak operator topology. Then, for all x,y∈H we have that ⟨Ti⁢x,y⟩→⟨T⁢x,y⟩. Thus, for all S∈ℱ, we have

⟨(T⁢S-S⁢T)⁢x,y⟩ = ⟨T⁢S⁢x,y⟩-⟨T⁢x,S*⁢y⟩
= lim⁡(⟨Ti⁢S⁢x,y⟩-⟨Ti⁢x,S*⁢y⟩)
= lim⁡⟨(Ti⁢S-S⁢Ti)⁢x,y⟩
= lim⁡⟨(Ti⁢S-Ti⁢S)⁢x,y⟩
= 0

Hence, T⁢S-S⁢T=0, so that T∈ℱ′. We conclude that ℱ′ is closed in the weak operator topology. □
