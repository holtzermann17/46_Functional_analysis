subadditive


Suppose V is a vector space (over a field), and f is a function f:V→ℝ. Then f is subadditive if

f⁢(x+y)≤f⁢(x)+f⁢(y),x,y∈V.

Examples

  1. 1.

    Any linear function V→ℝ is subadditive.

  2. 2.

    If ∥⋅∥ is a norm on V, a≥0, then

    f⁢(x)=a+∥x∥

    is subadditive.

Properties

Suppose f is subadditive.

  1. 1.

    If f is positively 1-homogeneous, then f is convex.

  2. 2.

    The sum of two subadditive functions is subadditive.
