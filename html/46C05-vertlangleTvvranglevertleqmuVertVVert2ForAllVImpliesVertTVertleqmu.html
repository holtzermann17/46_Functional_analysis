Theorem.

Let H be a unitary space, T be a self-adjoint linear operator and μ≥0. If |⟨T⁢v,v⟩|≤μ⁢∥v∥2 for all v∈H then T is a bounded operator and ∥T∥≤μ.

Proof.

We will show that ∥T⁢v∥≤μ⁢∥v∥ for all v∈H. This is trivial if ∥T⁢v∥ or ∥v∥ is zero, so assume they are not. Let λ be any positive number.

∥T⁢v∥2 =⟨T⁢v,T⁢v⟩
=14⁢[⟨T⁢(λ⁢v+1λ⁢T⁢v),(λ⁢v+1λ⁢T⁢v)⟩-⟨T⁢(λ⁢v-1λ⁢T⁢v),(λ⁢v-1λ⁢T⁢v)⟩]
≤μ4⁢[∥λ⁢v+1λ⁢T⁢v∥2+∥λ⁢v-1λ⁢T⁢v∥2]
≤μ2⁢[λ2⁢∥v∥2+1λ2⁢∥T⁢v∥2]

Now if we put λ2=∥T⁢v∥∥v∥ we get ∥T⁢v∥2≤μ⁢∥T⁢v∥⁢∥v∥ hence ∥T⁢v∥≤μ⁢∥v∥. ∎

F. Riesz and B. Sz-Nagy, Functional Analysis, F. Ungar Publishing, 1955, chap VI.
