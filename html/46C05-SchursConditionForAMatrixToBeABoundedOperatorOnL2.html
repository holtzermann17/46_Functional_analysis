Schur’s condition for a matrix to be a bounded operator on l2


Theorem 0.1

Let B be a matrix defined on T×T for some countable set T. If there exists a positive number C such that

∑t∈T|b⁢(s,t)|<C⁢for all ⁢s   𝑎𝑛𝑑   ∑s∈T|b⁢(s,t)|<C⁢for all ⁢t,

then B is a bounded operator on l2⁢(T) with its operator norm ∥B∥ less than or equal to C.

  • Proof.

    Let x be a sequence in l2⁢(T). We have

    ∥B⁢x∥22 =∑s∈T|∑t∈Tb⁢(s,t)⁢x⁢(t)|2
    ≤∑s∈T[∑t∈T|b⁢(s,t)|⁢(|b⁢(s,t)|⁢|x⁢(t)|)]2
    ≤∑s∈T[(∑t∈T|b⁢(s,t)|)⁢(∑t∈T|b⁢(s,t)|⁢|x⁢(t)|2)]
    ≤C⁢∑s∈T∑t∈T|b⁢(s,t)|⁢|x⁢(t)|2
    ≤C⁢∑t∈T|x⁢(t)|2⁢∑s∈T|b⁢(s,t)|
    ≤C2⁢∑t∈T|x⁢(t)|2.

    Therefore we have ∥B⁢x∥2≤C⁢∥x∥2 for all x∈l2⁢(T), hence ∥B∥≤C. □
