properties of the transpose operator


In this entry, whenever V,W are normed vector spaces, ℬ⁢(V,W) denotes the algebra of bounded linear operators V⟶W.

Let X,Y,Z be normed vector spaces and X′,Y′,Z′ be their continuous dual spaces. Let T,S∈ℬ⁢(X,Y), R∈ℬ⁢(Y,Z) and λ∈ℂ.

0.0.1 Basic properties

  • •

    T′∈ℬ⁢(Y′,X′) and ∥T∥=∥T′∥.

  • •

    (λ⁢T)′=λ⁢T′.

  • •

    (S+T)′=S′+T′.

  • •

    (R⁢T)′=T′⁢R′.

  • •

    If T-1 exists and T-1∈ℬ⁢(Y,X) then (T′)-1∈ℬ⁢(X′,Y′) and (T′)-1=(T-1)′.

0.0.2 Miscellaneous properties

  • •

    If we endow X′ and Y′ with the weak-* topology then T′:Y′⟶X′ is continuous.

  • •

    T is an isometric isomorphism if and only if T′ is an isometric isomorphism.

  • •

    If T is compact (http://planetmath.org/CompactOperator) then T′ is also compact.

  • •

    If T′ is compact and Y is a Banach space, then T is also compact.
