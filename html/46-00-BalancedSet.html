balanced set


Definition [1, 2, 3, 4] Let V be a vector space over ℝ (or ℂ), and let S be a subset of V. If λ⁢S⊂S for all scalars λ such that |λ|≤1, then S is a balanced set in V. The balanced hull of S, denoted by eq⁡(S), is the smallest balanced set containing S.

In the above, λ⁢S={λ⁢s∣s∈S}, and |⋅| is the absolute value (in ℝ), or the modulus of a complex number (in ℂ).

0.0.1 Examples and properties

  1. 1.

    Let V be a normed space with norm ||⋅||. Then the unit ball {v∈V∣||v||≤1} is a balanced set.

  2. 2.

    Any vector subspace is a balanced set. Thus, in ℝ3, lines and planes passing through the origin are balanced sets.

0.0.2 Notes

A balanced set is also sometimes called circled [3]. The term balanced evelope is also used for the balanced hull [2]. Bourbaki uses the term équilibré [2], c.f. eq⁡(A) above. In [5], a balanced set is defined as above, but with the condition |λ|=1 instead of |λ|≤1.
