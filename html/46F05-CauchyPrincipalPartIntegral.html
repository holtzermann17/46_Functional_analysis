Cauchy principal part integral


Definition [1, 2, 3] Let C0∞⁢(ℝ) be the set of smooth functions with compact support on ℝ. Then the Cauchy principal part integral (or, more in line with the notation, the Cauchy principal value) p.v.⁡(1x) is mapping p.v.⁡(1x):C0∞⁢(ℝ)→ℂ defined as

p.v.⁡(1x)⁢(u)=limε→0+⁡∫|x|>εu⁢(x)x⁢𝑑x

for u∈C0∞⁢(ℝ).

Theorem The mapping p.v.⁡(1x) is a distribution of first order (http://planetmath.org/Distribution4). That is, p.v.⁡(1x)∈𝒟′⁣1⁢(ℝ).

(proof. (http://planetmath.org/Operatornamepvfrac1xIsADistributionOfFirstOrder))

0.0.1 Properties

  1. 1.

    The distribution p.v.⁡(1x) is obtained as the limit ([3], pp. 250)

    χn⁢|x|x→p.v.⁡(1x).

    as n→∞. Here, χ is the characteristic function, the locally integrable functions on the left hand side should be interpreted as distributions (see this page (http://planetmath.org/EveryLocallyIntegrableFunctionIsADistribution)), and the limit should be taken in 𝒟′⁢(ℝ). It should also be noted that p.v.⁡(1x) can be represented by a proper integral as

    p.v.⁡(1x)⁢(u)=∫0∞u⁢(x)-u⁢(-x)x,

    where we have used the fact that the integrand is continuous because of the differentiability at 0. In fact, this viewpoint can be used to somewhat vastly increase the set of functions for which this principal value is well-defined, such as functions that are integrable, satisfy a Lipschitz condition at 0, and whose behavior for large x makes the integral converge at infinity.

  2. 2.

    Let ln⁡|t| be the distribution induced by the locally integrable function ln⁡|t|:ℝ→ℝ. Then, for the distributional derivative (http://planetmath.org/OperationsOnDistributions) D, we have ([2], pp. 149)

    D⁢(ln⁡|t|)=p.v.⁡(1x).
